# Для тебе · сайт-листівка до Дня Матері

Одна сторінка — п'ять актів — м'які кінематографічні переходи. Mobile-first, налаштовано на iPhone 13 (390×844).

## Локальний перегляд

Просто відкрити `index.html` у браузері. Або через простий локальний сервер:

```powershell
# в директорії проєкту
python -m http.server 8000
# або
npx serve .
```

Потім відкрити `http://localhost:8000` на телефоні (через Wi-Fi: `http://<твоя-локальна-IP>:8000`).

## Куди вставити справжній контент

Усі плейсхолдери — у папці `assets/`. Підміна = просто покласти файл з тим самим іменем (або поміняти `src=` у HTML).

| `family.svg`         | Сімейне фото (ти, я, Максим, Бланко) | full-bleed |
| `og-image.svg`       | Прев'ю для iMessage / Telegram    | 1200×630       |

**Рекомендації:**
- Замінити `.svg` на `.jpg` (або `.webp`) і оновити `src=` в `index.html`
- Стиснути до **80–120 КБ** на фото (iLoveIMG, Squoosh)
- Розмір по довшій стороні: **1200–1600px** достатньо для retina на телефоні
- Сімейне фото краще горизонтальне, з простором по центру (текст «Ти — неймовірна» накладається знизу)

### Відео (опційне)

Покласти `assets/maksym.mp4`:
- 10–15 секунд
- Формат **H.264 MP4**, ≤ 5 МБ
- Без звуку (атрибут `muted` стоїть)
- Бажано вертикальне, 9:16

Якщо відео не потрібне — видалити секцію `<section class="act act--video">` цілком (≈ рядок 158 в `index.html`).

### Тексти

Усі рядки з підписом `<!-- TODO: -->` у `index.html`. Основні точки правки:

- **Акт 1**, рядок ~50: `data-text="Ця історія про одну дівчину…"` — головна вступна фраза
- **Акт 2**, рядки ~67–93: підписи під фото Бланко
- **Акт 4**, рядки ~120–148: підписи під фото Максима + інтро «Це був найкращий і найважчий рік»
- **Акт 5**, сертифікат: рядки ~190–210
  - `[НАЗВА КНИГАРНІ]` — назва магазину
  - `XXXX` — сума в гривнях
  - текст-присвята всередині лапок
  - `№ XXXX-XXXX` — код сертифіката або інструкція

## Колір настрою

Якщо хочеться змінити загальну палітру — `styles.css`, секція `:root` (рядок 7). Найважливіші змінні:

```css
--cream:     #f5ebd9;   /* теплий фон */
--gold:      #c9a064;   /* золоті акценти */
--ink:       #2c1f17;   /* основний текст */
--rose:      #c79292;   /* пильна троянда */
```

## Розгортання

### Vercel (найпростіше)

```powershell
npm i -g vercel
vercel
```

Готово — отримаєш URL виду `https://md26.vercel.app`.

### GitHub Pages

1. Створити публічний репозиторій
2. `git init && git add . && git commit -m "init" && git push`
3. У налаштуваннях репо → Pages → Source: `main`, folder: `/ (root)`
4. URL буде `https://<username>.github.io/<repo>`

⚠️ **Важливо:** обидва сервіси віддають через HTTPS — це потрібно для `navigator.share` та `navigator.clipboard`.

## Як це виглядає у sharing

Прев'ю в iMessage / Telegram береться з `og-image.svg`. Бажано замінити на справжній JPG 1200×630 — наприклад, кадр з Максимом з підписом.

```html
<meta property="og:image" content="assets/og-image.jpg" />
```

## Технічні деталі (на випадок дебагу)

- Snap-scroll: `y proximity` (м'який, без жорсткого «прилипання»)
- Анімації шанують `prefers-reduced-motion`
- Інтерсекційний обсервер запускає reveal'и при threshold 0.28
- Конфеті — canvas, ~165 частинок, тривалість ~3.5 сек
- Конверт відкривається rotateX(178°), потім автопрокрутка до сертифіката
- Все працює офлайн (крім Google Fonts) — для повного офлайну скачати шрифти локально
